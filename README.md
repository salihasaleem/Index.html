## Customization Guide

### Update Business Information
Edit these sections in `index.html`:
- Location: Find "123 Sweet Street" and replace with your address
- Hours: Update opening and closing times
- Contact: Change phone number and email

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #FF6B9D;      /* Main color */
    --secondary-color: #C44569;    /* Secondary color */
    --accent-color: #FFA502;       /* Highlight color */
    --light-bg: #FFF9E6;           /* Light background */
}
```

### Add More Menu Items
Copy a `.menu-item` div in the Menu section and customize:
```html
<div class="menu-item">
    <div class="item-icon">🍨</div>
    <h3>Your Item Name</h3>
    <p>Description here</p>
    <ul class="flavors">
        <li>Flavor 1</li>
        <li>Flavor 2</li>
    </ul>
</div>
```

## Deploy Your Website

### Option 1: GitHub Pages
1. Go to your repository Settings
2. Scroll to "Pages" section
3. Select "main" branch as source
4. Your site will be available at your custom domain

### Option 2: Custom Domain
1. Purchase a domain from a registrar (GoDaddy, Namecheap, etc.)
2. Point domain to GitHub Pages
3. Add domain to repository settings

### Option 3: Other Hosting
- Upload files to any web hosting service
- Use services like Netlify, Vercel, or Firebase Hosting
